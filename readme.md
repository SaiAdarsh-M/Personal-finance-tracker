# Personal Finance Tracker (CLI)

A command-line based **Personal Finance Tracker** built using **Python** that allows users to record income and expenses, store transactions persistently using file handling, calculate current balance, and view transaction history through a menu-driven CLI application.

## How to Run

2. Navigate to the project directory  
   cd personal-finance-tracker

3. Run the program  
   python main.py
